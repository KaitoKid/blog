Back for the Summer - 2018 Edition

Posted on 2018-05-21

And I’m Back

Honestly, I don’t even know if this counts as being back. My previous blog post was just about a year ago where I just learned about how to setup and use Hexo. To be completely honest, I actually lost the files for that blog - I had to copy-paste everything from the static website over and regenerate the files from markdown.

In the end, I even removed the second how-to post because it wasn’t really relevant. Part 3 was supposed to be about how to get HTTPS on our own domain with GitHub pages, but now that GitHub has added that functionality… there’s really no point for part 3 either.

So yea, here I am. Essentially with a fresh start. Hopefully I’ll be able to be more consistent this time around with this blog and see where I can go. I’m expecting myself to go pretty ham for the first couple of days or weeks, and eventually slow it down to maybe one post a month.

Why I’m Blogging

I’ve been considering about it for a while, and it’s just something I’ve considered since middle school when a relatively close friend, Chelsey, used to blog. I say relatively because I didn’t know her super well back then, but now she’s one of my bestest friends. If you’re reading this Chelsey, thanks <3

Over the years, I’ve started blogs up, taken them down, and all that jazz but I’ve never actually dedicated myself to it - or seen the need to. It’s just like that one small project where in the beginning you’re super excited about it, but as time goes on, you realize how truly lazy you are and forget about it.

This time around though, it might be different. I’m mainly using this as an outlet as like a diary or a scrapbook, just a lot more messy and uncoordinated. I don’t intend to SEO, monetize, or even promote this blog. I probably won’t even list it in any of my social media profiles - I kinda just ended up on this domain because it’s where my Google account/email is grandfathered. I’ll be sharing my portfolio if anyone asks for my ‘website’.

To close it off for this post, I’m just going to talk about what’s been going on with my life. I guess it’s a lot longer than a closing should be, but I’m edgy like that so let’s kick it off with….

What’s Hexo?

Hexo is a static website generator that runs on NodeJS. It has a large community behind the project is a popular choice for blogs and personal websites hosted on GitHub pages. Unlike Jekyll, a similar generator, Hexo supports plugins in its themes.

Unlike other competitors like Wordpress, Ghost, and Medium, Hexo allows you to easily modify the HTML and Javascript. This flexibility was what prompted me to eventually choose Hexo myself.
